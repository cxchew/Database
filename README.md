# Database (SECP2523) 📊💾

Welcome to my portfolio repository for **SECP2523: Database**. This repository documents my journey in mastering relational database lifecycle mechanics, ranging from abstract semantic data planning to structural DDL implementation and advanced SQL analytics optimization.

The core focus of this course is understanding how to model complex real-world data environments, enforce structural operational integrity, eliminate storage redundancy anomalies through formal normalizations, and retrieve high-fidelity analytical insights using Structured Query Language (SQL).

## 🛠️ Core Technological & Theoretical Toolkit

* **Database Design & Architecture:** Conceptual Modeling, Logical Engineering, Data Dictionaries, Structural Mapping, User Interface (UI) Blueprints.
* **Normalization & Optimization Frameworks:** Data Redundancy Resolution, Modification Anomaly Mitigation, 1NF, 2NF, 3NF, Boyce-Codd Normal Form (BCNF).
* **SQL Language Commands (DDL & DML):** Relational Schema Tables (`CREATE TABLE`, `ALTER TABLE`), Integrity Constraints (`PRIMARY KEY`, `FOREIGN KEY`, `ON DELETE CASCADE`).
* **Advanced Query Syntax Processing:** Intersectional Lookups (`INNER JOIN`), Mathematical Summarizations (`SUM`, `AVG`, `ROUND`), Temporary Evaluation Strings (`CURDATE()`, `INTERVAL`).
* **Database Management Software (DBMS):** Microsoft Access Workspace Environments, Relational SQL Script Processing Platforms.
